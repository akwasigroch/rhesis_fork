# Deployment

Deploy Rhesis in various environments and configurations.

## Deployment Options

Choose the deployment method that best fits your needs:

- **[Self-Hosting](self-hosting)** - Deploy and manage your own Rhesis instance
- **[Docker Compose](docker-compose)** - Detailed Docker deployment guide  
- **[Production](production)** - Best practices for production environments

## Quick Start

For most users, we recommend starting with our Docker Compose setup for local development and testing, then moving to a production deployment when ready.
