# Getting starting with Rhesis SDK



The Rhesis SDK enables developers to create and manage tests sets for their Gen AI applications.
It also provides the functionality to manage tests on our platform.

## Installation

```bash
pip install rhesis-sdk
```

## Getting Started

The SDK can be used standalone or integrated with our platform. 
To cumminicate with our platform and to use LLMs provided by use you need to obtain an API key.

<Steps>
### Obtain an API key

1. Visit [Rhesis App](https://app.rhesis.ai)
2. Sign up for a Rhesis account
3. Navigate to your account settings
4. Generate a new API key

Your API key will be in the format `rh-XXXXXXXXXXXXXXXXXXXX`.
Keep this key secure and never share it publicly.

### Configure the SDK

You can configure the SDK either through environment variables or direct configuration:

<Tabs items={['Environment Variables', 'Direct Configuration']}>
<Tabs.Tab>
```bash
export RHESIS_API_KEY="your-api-key"
export RHESIS_BASE_URL="https://api.rhesis.ai"
```
</Tabs.Tab>
<Tabs.Tab>
```python
import rhesis
rhesis.api_key = "your-api-key"
rhesis.base_url = "https://api.rhesis.ai"  # optional
```
</Tabs.Tab>
</Tabs>


</Steps>
